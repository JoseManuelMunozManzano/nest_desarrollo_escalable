# 05-ws-client

Es la parte cliente de teslo-shop para el WebSocket que tenemos implementado.
Hecho en Vanilla JavaScript con TypeScript.

Proyecto creado con:

```
yarn create vite
```

Luego accedemos a la carpeta e instalamos las dependencias de vite:

```
yarn
```

Para ejecutar:

```
yarn dev
```
