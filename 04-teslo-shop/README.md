# Teslo API

Proyecto creado con:

```
nest new 04-teslo-shop
```

1. Levantar la base de datos

```
docker-compose up -d
```

Se ha visto:

- TypeORM
- Postgres
- CRUD
- Constrains
- Validaciones
- Búsquedas
- Paginaciones
- DTOs
- Entities
- Decoradores de TypeORM para entidades
- Métodos BeforeInsert, BeforeUpdate
